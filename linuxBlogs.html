Simple Commands that Will Make You Feel Like a Pro

November 28, 2015           Linux

Mastering simple commands can significantly enhance your productivity and efficiency. Here, we delve into some of the most powerful yet straightforward commands that can make you feel like a pro.

Basic Terminal Commands

If you know basic commands to enter in terminal, you will be more comfortable:

  • ls - Lists the contents of a directory.
    Example: ls /home/user lists all files and directories in /home/user.
  • cd - Changes the current directory.
    Example: cd /var/log changes the current directory to /var/log.
  • pwd - Prints the current working directory.
    Example: pwd might output /home/user, showing where you are in the filesystem.
  • mkdir - Creates a new directory.
    Example: mkdir projects creates a new directory called projects.
  • rm - Removes files or directories.
    Example: rm old_file.txt deletes the file old_file.txt. rm -r old_directory deletes the directory old_directory and its contents.
  • cp - Copies files or directories.
    Example: cp document.txt /backup/document.txt copies document.txt to the /backup directory. cp -r source_directory destination copies a directory and its contents.
  • mv - Moves or renames files or directories.
    Example: mv report.txt archive/report.txt moves report.txt to the archive directory. mv old_name new_name renames a file or directory.
